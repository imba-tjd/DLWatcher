DLsite Lowest Price Watcher

For a long time there isn't a DLsite version of SteamDB. It's easy to create one. But it's hard to:

  • not abuse crawling
  • make good use of data
  • prevent from being abused

This program retrieves data from https://www.dlsite.com/<category>/ranking/total. Namely it only tracks best sellers. You're not able to trivially modify this to track custom artifact.

Recording starts from 2022/4/3
